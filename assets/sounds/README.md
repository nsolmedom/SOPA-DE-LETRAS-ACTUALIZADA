# Sound Effects

Please place the following MP3 files in this directory (`sounds/`):

- `click.mp3` - For button clicks
- `select.mp3` - For selecting letters in the grid
- `correct.mp3` - When a word is found
- `wrong.mp3` - When a selection is invalid
- `level_complete.mp3` - When a level is finished

The game will automatically load and play these sounds if they exist.
